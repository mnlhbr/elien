Auf einem Linuxsystem muss man folgende Programme und Bibliotheken installieren:

  • SDL2
  • glew
  • glm
  • lua5.2
  • git

Es muss ein aktueller C++ Kompiler verwendet werden da C++11 benutzt wird. Desweiteren benötigt man verschiedene Helfer wie make, die aber auf jedem Linuxsystem vorhanden sein sollten.

$ git clone https://github.com/madowlet/elien.git
$ cd elien
$ make -j
$ bash run.sh
